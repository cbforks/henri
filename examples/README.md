
# henri example folder

## How to use

Clone the [repo](https://github.com/simplehub/henri.git):

```
git clone https://github.com/simplehub/henri.git
```

Install it and run:

```bash
cd henri
npm install
cd examples/<example here>
npm install
npm run dev
```
